Question : Describe the scope of var, let and const

Answer : var variable is a functional scope also called a global variable. It can be re-declare and re-assaign from anywhere. var can be accessed but when its value contain nothing it will show undefind. The scope of a let variable is block scope. Block scope means inside the {} curly bracket. It's value can be re-assign but it can't be re-declare. Otherhand The scope of a const variable also a block scope. But const can't re-declare or re-assaign. It cannot be declared without initialization

Question : Tell us the use cases of null and undefined

Answer : Two types of datatype in javaScript (primitive and non-primitive). null and undefined both are both primitive data type. When a variable is declared but has not yet been assigned a value it will return undefined. But null is a declared value. That return a null when it's value was not found.
